**Что применено**

В первой части дипломного проекта применено:
1. Адаптивная верстка, cвестано для 4 разрешений:
  * 1280px;
  * 1024px;
  * 768px;
  * 320px;
2. React JSX;
3. Flexbox;
4. CSS Grid Layout.


**[Ссылка на проект](https://project.movies.nomoreparties.co/)**
